# batch_renamer_v2

파일 및 폴더 이름을 일괄적으로 변경하고, 텍스트 및 엑셀 파일 내부의 내용까지 검색 및 변경할 수 있는 도구입니다.

## 주요 기능
- **일괄 이름 변경**: 대량의 파일/폴더 이름을 한 번에 변경
- **내용 검색 및 변경**: 텍스트(.txt, .csv, .log) 및 엑셀(.xlsx, .xls, .xlsm, .xlsb) 파일 내 문구 검색/변경
- **상세 미리보기**: 변경 전 상세 내용을 확인하고 특정 항목만 선택하여 변경 가능
- **중단 기능**: 작업 중 언제든지 안전하게 중단(Stop) 가능
- **사용 설명서**: 프로그램 내 도움말 버튼 포함

## 다운로드
[여기에서 batch_renamer_v2.exe를 다운로드하세요](https://github.com/saramida9205/renamer/raw/master/releases/batch_renamer_v2.exe)

## 사용 방법
1. `batch_renamer_v2.exe`를 실행합니다.
2. 검색할 경로를 설정합니다.
3. 검색할 키워드와 변경할 키워드를 입력합니다.
4. '검색하기' 버튼을 눌러 결과를 확인합니다.
5. '일괄 이름 변경' 또는 상세 미리보기의 '변경 실행'을 통해 작업을 완료합니다.
