# batch_renamer_v3

파일 및 폴더 이름을 일괄적으로 변경하고, 텍스트 및 엑셀 파일 내부의 내용까지 검색하여 변경하거나 **일괄 삭제**할 수 있는 강력한 도구입니다.

## v3 주요 변경 및 개선 사항
- **선택 항목 일괄 삭제 (New)**: 검색 결과에서 파일 자체 또는 파일 내부의 특정 내용(줄/셀)을 선택하여 한 번에 삭제할 수 있습니다.
- **전화번호 검색 강화**: 다양한 형식의 휴대전화 번호(`010-1234-5678`, `010 1234 5678`, `01012345678`)를 감지하며, **파일명 자체가 전화번호인 경우**도 검색 및 처리가 가능합니다.
- **안전 종료 로직**: 프로그램 종료 시 진행 중인 작업을 확인하고 안전하게 종료할 수 있도록 개선되었습니다.
- **UI 개선**: 버튼 배치 최적화 및 메인 화면에서도 일괄 삭제가 가능하도록 기능을 확장했습니다.

## 주요 기능
- **일괄 이름 변경**: 대량의 파일/폴더 이름을 한 번에 변경
- **내용 검색 및 변경**: 텍스트(.txt, .csv, .log) 및 엑셀(.xlsx, .xls, .xlsm, .xlsb) 파일 내 문구 검색/변경
- **상세 미리보기**: 변경 전 상세 내용을 확인하고 특정 항목만 선택하여 정밀 변경 가능
- **중단 기능**: 작업 중 언제든지 안전하게 중단(Stop) 가능

## 다운로드
[여기에서 batch_renamer_v3.exe를 다운로드하세요](https://github.com/saramida9205/renamer/raw/master/releases/batch_renamer_v3.exe)

## 사용 방법
1. `batch_renamer_v3.exe`를 실행합니다.
2. 검색할 경로를 설정합니다.
3. 검색할 키워드 또는 **'전화번호 검색'** 옵션을 선택합니다.
4. '검색하기' 버튼을 눌러 결과를 확인합니다.
5. '일괄 이름 변경' 또는 **'선택 항목 일괄 삭제'**를 통해 작업을 완료합니다.
